# Fastype

A 2D game for practising fast typing

## Technologies used

C++, SFML


## Getting started

This is how you get the application to work on linux

### Prerequisities

```
You need to have SFML installed on your machine
```

### Running

```
Linux: Clone the repository and from terminal just run:
make fastype
./fastype
```
